# 📄 Gerador de Currículo Profissional

Um gerador de currículos moderno, intuitivo e profissional desenvolvido com **Vue 3**, **Vite** e **Tailwind CSS**. Crie currículos de alto impacto em minutos com pré-visualização em tempo real e diversos modelos customizáveis.

## 🚀 Funcionalidades

- **✨ Pré-visualização em Tempo Real**: Veja as alterações no seu currículo instantaneamente enquanto preenche o formulário.
- **📑 Formulário em Etapas**: Interface organizada em passos lógicos (Pessoal, Experiência, Educação, Habilidades e Design).
- **🎨 Modelos Profissionais**: Escolha entre 5 templates exclusivos (Moderno, Clássico, Minimalista, Elegante e Criativo).
- **🖱️ Drag-and-Drop**: Reordene suas experiências e formações facilmente arrastando os itens.
- **📱 Responsivo**: Interface adaptada para diferentes tamanhos de tela.
- **💾 Persistência Local**: Seus dados são salvos automaticamente no navegador para que você não perca nada.
- **📥 Exportação em PDF**: Gere um PDF de alta fidelidade pronto para impressão ou envio.
- **📞 Máscara de Telefone**: Formatação automática inteligente para números de telefone brasileiros.

## 🛠️ Tecnologias Utilizadas

- **[Vue 3](https://vuejs.org/)**: Framework progressivo para construção de interfaces.
- **[Vite](https://vitejs.dev/)**: Ferramenta de build ultra-rápida.
- **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS utilitário para design moderno.
- **[Lucide Vue Next](https://lucide.dev/)**: Conjunto de ícones bonitos e consistentes.
- **[html2pdf.js](https://ekoopmans.github.io/html2pdf.js/)**: Conversão de HTML para PDF com alta fidelidade.
- **[Vuedraggable](https://github.com/SortableJS/Vue.Draggable)**: Componente para listas ordenáveis via arrasto.

## 📦 Como rodar o projeto

1. **Clone o repositório**
   ```bash
   git clone https://github.com/Lucas19Alves/gerador-curriculo.git
   ```

2. **Instale as dependências**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento**
   ```bash
   npm run dev
   ```

4. **Acesse no navegador**
   O projeto estará disponível em `http://localhost:5173` (ou na porta indicada no terminal).
